A component can't change from uncontrolled input to be controlled or vise versa. This is likely caused by the value changing from undefined to a defined value or vise versa, which should not happen. Decide between using a controlled or uncontrolled input element for the lifetime of the component, Input elements must be either controlled or uncontrolle and not both (XOR) , by default component is uncontrolled if we don't specify value or defaultValue
